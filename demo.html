üîß How to Use

To add this topbar to your restaurant website, simply include this script tag in your HTML:

<script src="path/to/restaurant-hours-topbar.js"></script>

Customization

Edit the RESTAURANT_CONFIG object at the top of the script to customize:

  • Restaurant name (currently not displayed but available for future use)
  • Timezone - Set your restaurant's timezone
  • Opening hours - Set different hours for each day
  • Timing thresholds - When to show "closes soon" vs "opens soon"
  • Styling - Colors, fonts, and sizes

Technical Notes

  • The script is wrapped in an IIFE to avoid global scope pollution
  • It checks for existing topbars to prevent duplicates
  • Uses sessionStorage to remember dismissal
  • Automatically adjusts body padding to prevent content overlap
  • High z-index (999999) ensures it stays on top
  • Responsive design that hides detailed info on mobile
